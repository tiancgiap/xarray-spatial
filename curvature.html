Curvature¶

xrspatial.curvature.curvature(raster)[source]¶

Compute the curvature (second derivatives) of a raster surface.

Parameters

raster (xarray.DataArray) – 2D input raster image with shape=(height, width)

Returns

curvature – Curvature image with shape=(height, width)

Return type

xarray.DataArray

Examples

>>> raster=np.array([[1, 1, 0, 2],[0, 2, 1, 2]])
>>> temp_raster = xarray.DataArray(raster, dims=['y', 'x'])
>>> xrspatial.curvature(temp_raster)
Out[]:
<xarray.DataArray (y: 2, x: 4)>
array([[  -0.,  100., -300.,   -0.],
   [  -0.,  300., -200.,   -0.]])
Dimensions without coordinates: y, x
